# Ghostty Configuration

Minimal, distraction-free terminal setup with a Tokyo Night colour scheme.

## Appearance

- **Font:** Iosevka Nerd Font at 14px with ligatures enabled
- **Theme:** Tokyo Night — dark blue-grey background (`#1a1b26`) with a balanced 16-colour ANSI palette spanning muted greys, vibrant blues, pinks, greens, and oranges
- **Opacity:** 95% background transparency
- **Cursor:** Blue block (`#7aa2f7`), auto-hides on keystroke

## Window

Decorations are disabled for a clean look. Zero padding, balanced padding enabled, and step-resize active. The resize overlay is suppressed entirely. GTK wide tabs are off.

## Behaviour

- Clipboard: manual copy/paste only (no copy-on-select)
- Scrollback: 10 000 lines
- Shell integration: automatic detection
- Image storage capped at 100 MB

## Keybindings

| Binding | Action |
|---|---|
| `Ctrl+Shift+C` | Copy |
| `Ctrl+Shift+V` | Paste |
| `Alt+Enter` | Toggle fullscreen |
